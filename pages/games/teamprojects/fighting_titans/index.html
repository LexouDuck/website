Fighting Titans

This is a game I worked on with a buddy from the Spriters Resource forums, Sketchasaurus. He made pretty much all of the graphics, and thought out most the gameplay concepts. I did all of the programming, as well as pitching in ideas and designs. It’s a sort of fighting game in "fake 3D", or what some call "2.5D", built according to NES (Nintendo Entertainment System) console limitations.

I programmed the game in Haxe, using the Haxeflixel library.

The game features 8 playable “titan” characters, each with different stats, projectile attacks, and special moves.
